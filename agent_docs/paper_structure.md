# Paper Structure

## Target venue
NeurIPS 2026 Workshop (or main conference if results are strong enough)

## Sections
1. **Abstract** (150 words max)
2. **Introduction** — FL on decentralized compute, threat model, contribution summary
3. **Background** — Byzantine FL, backdoor attacks, Akash trust model
4. **Threat model** — attacker capabilities, honest majority assumption, reputation gaming
5. **Experimental setup** — ByzFL simulation, datasets (CIFAR-10, GPT-2), metrics
6. **Results** — baseline defenses vs attacks, scalability, Akash overhead
7. **Reputation-weighted defense** — algorithm, trust score design, combined approach
8. **Discussion and limitations** — reputation gaming, Sybil resistance, generalization
9. **Related work** — Krum, TrMean, FLTrust, LASA, DeTrigger, Covenant-72B
10. **References**
